core_runner.handler.local_start_execution

core_runner.handler.local_start_execution(task_payload: TaskPayload, name: str) → dict[source]

Start the execution of a step function in local mode. This means it will run in a shell process in the background and continue to run until it fails, completes, or is stopped by the OS.

Parameters:

  • task_payload (TaskPayload) – The task payload to execute

  • name (str) – the executin name

Returns:

Result of the job start request

Return type:

dict
