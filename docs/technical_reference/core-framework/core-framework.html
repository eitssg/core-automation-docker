core-framework

This is the core framework of the application. This code provides data models, constants and tools that provide consistency across all modules of the appliction.

You may find the source for thie core-framework on GitHub at here https://github.com/eitssg/core-framework.

Constants

core_framework.constants

This module provides a list of constants that are used throughout the Core-Automation framework.

core_framework.status

This module provides the BuildStatus class which is used to track the status of a build.

Object Models

core_framework.models

This module provides all of the data models that are used throughout Core-Automation.

Common Functions

core_framework.common

The common module provides a suite of function that are used throughout the Core Automation framework.

core_framework.prn_utils

Utilities to validate or generate PRN Identifiers

Local Execution Magic

core_framework.magic

Magic module for emulating the boto3 S3 client and Buckets so we can elect to store files locally instead of in S3
