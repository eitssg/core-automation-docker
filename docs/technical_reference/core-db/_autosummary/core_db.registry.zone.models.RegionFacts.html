Region FACTS descriging the detailed information for each supported region in the Zone

Region FACTS are provided for each region Alias available in the zone. THe region alias is a slug name for the region.

Examples Include:

  • us-west-1 -> usw

  • us-east-1 -> use

  • ap-southeast-1 -> sin

  • ap-southeast-2 -> syd

When registering RegionFacts, you may do so under any “slug” name you wish as they are user defined and scopeed to the Zone.

Regions are created by Network Administrators. You must have the role “CoreNetworkAdmin” to be able to modify the RegionFacts.
