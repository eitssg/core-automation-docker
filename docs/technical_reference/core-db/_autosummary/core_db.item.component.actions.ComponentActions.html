classmethod patch(**kwargs) → Response

Allows you to specify only a partial set of fields to update in a record.

The effect will be that the full record will be fetched, the fields specified will be updated, and the full record will be saved back to the table.

Some implementations may use the dynamodb PATCH operation to update a field.
